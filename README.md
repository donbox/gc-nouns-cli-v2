# gc-nouns-cli-v2

Focused design workspace for the next CLI v2 effort beyond packs and
registries.

## Scope

This repo is for working through the command surface around the other major GC
nouns, including:

- `city`
- `rig`
- `session`
- `agent`
- `formula`
- `order`
- `skill`
- `mcp`

## Contents

- `design/nouns-cli-unification.md`
  - the current design note for the non-pack noun surface

## Intent

This repo is intentionally lightweight. It exists so the design work can be
reviewed in isolation before any implementation or larger docs changes.
